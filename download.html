Downloading Ramen

Docker

There is a docker image available as rixed/ramen on https://hub.docker.com. This image also embeds collectd and fprobe to help getting data in quickly. You can decide not to run those and run only the daemons implementing Ramen with the --only option.

Debian packages

Sources

You can also build from sources on either Linux (recommended) or MacOS. See building from sources.
