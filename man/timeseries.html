ramen-timeseries

NAME

Ramen-timeseries - Extract a timeseries from an operation

SYNOPSIS

Ramen timeseries [OPTION]... OPERATION FIELD...

ARGUMENTS

FIELD (required)

Fields to retrieve values from

OPERATION (required)

Operation unique name

OPTIONS

--bucket-time=begin|middle|end (absent=begin)

Selected bucket time

--consolidation=min|max|avg|sum (absent=avg)

Consolidation function

-f FIELD, --factor=FIELD

specify which fields to use as factors/categorical variables

-h, --with-header, --header

Output the header line in CSV

--help[=FMT] (default=auto)

Show this help in format FMT. The value FMT must be one of ‘auto', ‘pager', ‘groff' or ‘plain'. With ‘auto', the format is ‘pager‘ or ‘plain' whenever the TERM env var is ‘dumb' or undefined.

-n POINTS, --num-points=POINTS (absent=0)

Number of points returned

--null=VAL (absent=<NULL> or RAMEN_CSV_NULL env)

Representation of NULL values

--pretty

Prettier output

--separator=VAL (absent=, or RAMEN_CSV_SEPARATOR env)

Field separator

--since=SINCE

Timestamp of the first point

--time-step=DURATION (absent=0.)

Duration between two points

--timeout=VAL (absent=300.)

Operation will stop archiving its output after that duration if nobody ask for it

--until=UNTIL

Timestamp of the last point

--version

Show version information.

-w FIELD=VALUE, --where=FIELD=VALUE

Output only tuples which given field match the given value

COMMON OPTIONS

-d, --debug (absent RAMEN_DEBUG env)

Increase verbosity

--initial-export-duration=VAL (absent=0. or RAMEN_INITIAL_EXPORT env)

How long to export a node output after startup before a client asks for it

--persist-dir=VAL (absent=/tmp/ramen or RAMEN_PERSIST_DIR env)

Directory where are stored data persisted on disc

-q, --quiet (absent RAMEN_QUIET env)

Decrease verbosity

-S, --keep-temp-files (absent RAMEN_KEEP_TEMP_FILES env)

Keep temporary files

--seed=VAL, --rand-seed=VAL (absent RAMEN_RANDOM_SEED env)

Seed to initialize the random generator with (will use a random one if unset)

--variant=VAL (absent RAMEN_VARIANTS env)

Force variants

See option --persist-dir.

See option --quiet.

See option --seed.
